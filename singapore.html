Skyscrapers are a common sight in Singapore. First-time visitors to Singapore will be amazed by our business district and residential areas. We are known to the world as a "Garden City", so do explore places like the Botanical Gardens, and Night Safari.
Best Airport - Changi Airport. Top airline - Singapore Airline. We are a small city-country. But we aim to be the BEST to compete at global standards.
Singapore is summer all year round, 365 days. However occasion rain can help to keep the citizens cool. Humidity is high at 95%, compared to other regions like America or Europe.


How did I do this?

For this page, I have added shadow for each of these boxes. The shadow effect is a nice touch to bring focus to your contents. You can added shadow easily by learning CSS. Try this online tool here.

Inside each box, the left right alignment is achieved by using CSS flexbox. You can see that the layout stay nicely even if you resize the web browser window. Trying resizing to a tablet size, or even a mobile phone screen size. Flexbox properties can handle both "grow" and "shrink" for the width and height of the HTML elements.

I have added CSS codes - media queries to handle the different screen sizes. This is taught in class on the chapter "Responsive Web Design".
