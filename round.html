How did I do this?

This above is done with flexbox too. Once I set CSS property "flex-wrap" to "wrap", if the window is resized to a smaller one, the row of 4 becomes row of 2, then 1 each for the smallest viewport. Try resizing the window now.

I learned how to make round images too. The CSS property "border-radius" with value at 50%, can create such results. And not forgetting the img tag with CSS property "object-fit" must be set to "cover".

Navigation Bar
