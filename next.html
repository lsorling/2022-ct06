How did I do this?

I chose thick borders for this set of photos to enhance the contents of the images. Orange color is selected as it matches most of the photos primary color. Look at the photos carefully. Did you notice it?

This is done with flexbox too. Once I set CSS property "flex-wrap" to "wrap", if the window is resized to a smaller one, the row of 4 becomes row of 2, then 1 each for the smallest viewport. Try resizing the window now.

I learned how to make rounded corners for the box. The CSS property "border-radius" with value at 35px, can create such results. And not forgetting the img tag with CSS property "object-fit" must be set to "cover".
